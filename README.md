#Aplicación para crear las tablas de multiplicar.
```
Opciones:
      --help     Muestra ayuda                                        [booleano]
      --version  Muestra número de versión                            [booleano]
  -b, --base     Es la base de la tabla de multiplicar que se genera
                                                            [número] [requerido]
  -l, --listar   Muestra por consola la tabla generada
                                                   [booleano] [defecto: "false"]
  -h, --hasta    Es el limite hasta el que se va a llegar [número] [defecto: 10]
```
